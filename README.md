# 💙 Bluepir-Madness
A Balatro mod for Bluepirs and her angelic community.

## Background
This mod, starting as [a single joker made as a test](https://github.com/HonestParley27/Bluepir-Madness/tree/Original-Mod-Precursor), evolved into a full-fledged play modification by JustNotSebas and HonestParley27 as a gift for Bluepirs, both for her birthday ~~(done in a time crunch)~~ and as a memory to her amazing community, which has been a huge inspiration for most stuff in this mod.

If you don't know her, check out Blue here!

- Twitch: https://www.twitch.tv/Bluepirs
- VGen: https://vgen.co/Bluepirs
- Discord: https://discord.gg/wjM5MTEGbx

## How to install?
0. Remember that your game **must** have already been modded with Lovely and Steammodded. If you haven't, [you can check out zZMaDsKILLsZz's guide on Steam here.](https://steamcommunity.com/sharedfiles/filedetails/?id=3400691352)

1. [Download the latest release of the mod here.](https://github.com/HonestParley27/Bluepir-Madness/releases/latest) Click "Source code (zip)" as download option.

2. Drop the zip file in the Mods folder in Balatro. Refer to the linked guide to get the exact location for your system.

3. Open the game and have fun!
